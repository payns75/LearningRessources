## Introduction
CTRL+o = recherche une note par titre
CTRL+n = créer une nouvelle note
CTRL+P = liste des commande d'Obsidian
CTRL+G = vue graphique
CTRL+ALT Fleche gauche ou droite pour naviger dans les notes rapidement.

Canvas, pas souvent utilisé, à regarder éventuellement.
Note quotidienne ajouter le raccourci CTRL+d qui n'existe pas. La note quotidienne est beaucoup utilisée.
Ajouter des # (tag), les titres commencent à partir du double #. Les tags sont visibles dans le panel de gauche.

## Commandes pour écrire dans Obsidian.
- CTRL+I mettre en italique simple *
- CTRL+B mettre en gras double *
- Surligner double =
- CTRL+P code utiliser le ``
- CTRL+P Strike pour barrer ou utiliser le tild ~~
- Équation, mettre entre deux $: $e^{\pi i} + 1 = 0$
- Todo: un tiret espace un crochet espace
- Listes numérotés: 1. etc...
- Séparateurs:
    - Triple -
    - Triple *
    - Triple _
- Pour faire un lien vers une note: double crocher [[]]
    - Il est possible de faire un lien vers une note et directement vers une section de titre
    - Pour prévisualiser le contenu de la note, il suffit d'ajouter le point d'exclamation.

## Insert template
Pour insérer un template CTRL+P Insert template puis choisir le template.
